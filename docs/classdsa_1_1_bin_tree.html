二叉树类 

(1) 基本概念
--------------— r      ---—
                 /depth(v)        .                *
               / \
-------—    v  ...      height(r)
             / \
height(v)   .-------—  .
           / \             .           ---—
v的深度：depth(v)，指子树根节点v的深度
根节点深度为0
v的高度：height(v)，指子树根节点v的高度
空树高度取为-1
叶子节点：没有后代(子节点)的节点
(2) 二叉树：节点度数<=2的树
对于深度为k的节点，数量至多为2^k个；
含n个节点，高度为h的二叉树： h < n < 2^(h+1)
n = h+1时，树成为单链
n = 2^(h+1)-1时，二叉树为满树
