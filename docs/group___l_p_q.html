合并左式堆 

a和b为两个左式堆的根节点：
  a      b
 / \    / \
al ar  bl br
将a的右子堆ar与b合并，合并之后的结果AR仍做为a的右子堆
   a
 /   \
al   AR
   ------------
   |      b   |
   | ar  / \  |
   |    bl br |
   -----------
比较al与AR的npl值，通过互换，使 npl(a->left) >= npl(a->right)

a,b待合并的两个左式堆
