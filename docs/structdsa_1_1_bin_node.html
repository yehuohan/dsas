层次遍历(LayerOrder) 

通过迭代实现：
            d
          /   \
        l1    r1
       / \
     l2  r2
层次遍历  : d - l1 - r1 - l2 - r2
将遍历的顺序，看成一个队列：d最先入队，故最先访问；
                            r2最后入队，最后访问；
