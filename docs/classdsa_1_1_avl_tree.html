详细描述

template<typename T>
class dsa::AvlTree< T >

Avl平衡二叉树类

继承自二叉搜索树(BST)，沿用BST的search函数； 需要借助avl平衡因子，实现平衡； search, insert, remove在最坏情况下需要O(logh)时间。
