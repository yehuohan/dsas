  带有R的节点为红，未做特别说明则为黑；

使用分裂修复上溢 


取中位数分裂，s = ⌊m/2⌋ (即向下取整，m为b-tree的阶)，如下图所示：
即m为奇数时，s为key正中间位置，m为偶数时，s为key均分后左侧的最后一个位置。
*[k0]*[k1]*[k2]* ... *[k(m-1)]*
                  |
                  V
*[k0]*[k1]*k(s-1)] * [ks] * [k(s+1)]* ... *[k(m-1)]*
------------------   ----   -----------------------
作为ks左子节点   上移至父节点         作为ks右子节点

将[3]上移至父节点

    r    \   s     =>        /       \
          \ /               /         \
 *[1]*[2]*[3]*[5]*     *[1]*[2]*    *[5]*
 -----------------     ---------    -----
    原node节点       分裂后的node   新节点rc
